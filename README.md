# Blood Cell Classification using DenseNet121

A deep learning application for classifying microscopic blood cell images using DenseNet121 and Transfer Learning.

## Features

- Blood Cell Classification
- DenseNet121 Transfer Learning
- Streamlit Web Application
- Confidence Score
- Top-3 Predictions

## Tech Stack

- Python
- TensorFlow
- Streamlit
- OpenCV
- Plotly

## Dataset

Blood Cells Image Dataset (Kaggle)

## Run Locally

```bash
pip install -r requirements.txt

streamlit run app.py
